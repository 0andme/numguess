# numguess

사용자로부터 이름과 1부터 100사이의 값을 입력 받고 이를 랜덤으로 생성된 정답과 비교

## Installation

```shell
$ pip install
```

## How to Start

```shell
$ python numguess.py
```

## Features

- Set answer (randomly 1 to 100)
- Get user name 
- Get user's guess
- Campare answer with user's guess
